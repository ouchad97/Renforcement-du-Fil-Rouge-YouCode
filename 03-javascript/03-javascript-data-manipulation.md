# JavaScript – Manipulation de données

## Objectif

Cette activité permet de pratiquer **la manipulation de données avec JavaScript** :

* parcours de tableaux
* filtrage des données
* tri
* recherche
* modification dynamique des éléments

Les résultats doivent être **affichés dans la console** ou appliqués aux **cartes dans la page**.

Utilisez cette page pour réaliser les exercices :

# 🧩 Data Manipulation Challenge — Série de 15 Missions

## 🟢 Niveau 1 : Exploration des données

1. Afficher **tous les noms dans la console**.
   → Fonction possible :
   `function showAllNames()`

2. Afficher uniquement les **nationalités (sans doublons)**.
   → Fonction possible :
   `function showUniqueNationalities()`

3. Compter combien de personnes sont **Marocaines** et loguer le résultat.
   → Fonction possible :
   `function countMoroccan()`

---

## 🟡 Niveau 2 : Manipulation du DOM

4. Créer un bouton qui **supprime toutes les cartes des Tunisiennes**.
   → Fonction possible :
   `function removeTunisians()`

5. Créer un bouton qui **colore en vert toutes les cartes des Marocains**.
   → Fonction possible :
   `function highlightMoroccan()`

6. Créer un bouton qui **trie les cartes par nom** et logue l’ordre dans la console.
   → Fonction possible :
   `function sortByName()`

7. Créer un bouton qui **filtre les Marocains seulement** et logue leurs noms.
   → Fonction possible :
   `function filterMoroccan()`

---

## 🟠 Niveau 3 : Recherche et filtrage

8. Afficher dans la console toutes les personnes **dont le prénom commence par "A"**.
   → Fonction possible :
   `function showNamesStartingWithA()`

9. Créer un **champ de recherche** qui logue les cartes correspondant au texte saisi.
   → Fonction possible :
   `function searchCards(text)`

10. Créer un bouton qui affiche **le nombre de cartes visibles** dans la console.
    → Fonction possible :
    `function countVisibleCards()`

---

## 🔴 Niveau 4 : Logique avancée

11. Créer un bouton qui **mélange les cartes (randomize)** et logue l’ordre dans la console.
    → Fonction possible :
    `function randomizeCards()`

12. Créer un bouton qui **réinitialise l’état initial des cartes** et logue la liste originale.
    → Fonction possible :
    `function resetCards()`

13. Calculer **la moyenne d’âge des personnes** et afficher le résultat dans la console.
    → Fonction possible :
    `function averageAge()`

14. Créer un **select / dropdown** pour filtrer par nationalité et afficher les résultats dans la console.
    → Fonction possible :
    `function filterByNationality(nationality)`

15. Permettre **d’éditer le nom et prénom des cartes** via un `prompt` ou un champ inline et loguer la nouvelle valeur.
    → Fonction possible :
    `function editName(cardId)`

---

## Compétences travaillées

* manipulation des **tableaux**
* `map()`
* `filter()`
* `find()`
* `sort()`
* manipulation du **DOM**
* gestion des **événements**
* logique JavaScript
