# JavaScript – Manipulation du DOM

## Objectif

L’objectif de cette activité est de pratiquer la **manipulation du DOM avec JavaScript** à travers une série de défis progressifs.

Vous allez interagir avec un escalier dynamique et modifier les éléments HTML en utilisant JavaScript.

## Ressource

Utilisez cette page pour réaliser les exercices :

https://ouchad97.github.io/e/

Ouvrez la page dans votre navigateur puis utilisez **JavaScript dans la console ou dans votre fichier JS** pour résoudre les missions.

---

# 🧩 Escalier DOM Challenge — Série de 15 Missions

## 🟢 Niveau 1 : Découverte du DOM

1. Change la couleur de la **première marche**.
2. Agrandis **toutes les marches de l’escalier**.
3. Change uniquement la couleur des **marches impaires**.
4. Ajoute un **texte personnalisé** sur la dernière marche.
5. Ajoute une **nouvelle classe** à la troisième marche.

---

## 🟡 Niveau 2 : Dynamique et interactions

6. Crée un **bouton** qui, lorsqu’on clique dessus, change la couleur de **toutes les marches**.

7. Crée un bouton qui **inverse la direction de l’escalier**
   (de **haut → bas** vers **bas → haut**).

8. Ajoute un bouton **“+”** qui ajoute **une nouvelle marche à chaque clic**.

9. Ajoute un bouton **“–”** qui supprime **la dernière marche**.

10. Crée une **barre de recherche** permettant de saisir le numéro d’une marche.
    Lorsque l’utilisateur valide, **la marche correspondante change de couleur**.

---

## 🟠 Niveau 3 : Animation et logique

11. Crée un bouton qui **anime les marches une par une**
    (effet d’apparition progressive).

12. Ajoute une interaction qui ne change que les marches :

* paires
* impaires
* ou contenant **une classe spécifique**

13. Crée un bouton qui **ajoute automatiquement une marche toutes les 2 secondes**
    jusqu’à atteindre **10 marches**.

14. Ajoute un bouton **“Cacher / Montrer”** qui rend l’escalier **visible ou invisible** sans le supprimer du DOM.

---

## 🔴 Niveau 4 : Créativité et gestion avancée

15. Active le **glisser-déposer (drag & drop)** pour permettre de **réorganiser librement les marches de l’escalier**.

---

## Compétences travaillées

* `querySelector` / `querySelectorAll`
* `classList`
* `addEventListener`
* manipulation des styles
* création et suppression d’éléments
* logique JavaScript
* animations simples
* interactions utilisateur
